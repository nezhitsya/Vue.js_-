## 개발환경 설정
- Chrome
- Visual Studio Code
    - 플러그인
        - Vetur
        - Night Owl
        - Material Icon Theme
        - Live Server
        - ESLint
        - Prettier
        - Auto Close Tag
        - Atom keymap
- Node.js
- Vue.js Devtools (Chrome 확장 프로그램)
- [Vue CLI](https://cli.vuejs.org/guide/)
    - 설치
    ```
    npm install -g @vue/cli
    ```
    - 프로젝트 생성 및 서버 실행
    ```
    vue create 프로젝트 명
    cd 프로젝트 명
    npm run serve
    ```
    - 프로젝트 제거
    ```
    rm -rf 프로젝트 명/
    ```
    - 서버 종료
    ```
    ctrl + c
    ```
- Vuex
    - 설치
    ```
    npm i vuex@3.6.2 --save
    ```
- Vue Router
    - 설치
    ```
    npm i vue-router@3.5.3 --save
    ```
